## ¿Qué es AR Atelier?
Una app móvil para que las mujeres encuentren, reserven y compren vestidos de fiesta sin salir de casa.
El proposito es que los usuarios puedan visualizar a detalle los vestidos con 3D,esto está potente...

---

## Tecnologías
| Stack | Por qué |
|-------|---------|
| **React Native 0.79** | Una sola base de código, dos tiendas de apps. |
| **Expo SDK 53** | Sin tocar Xcode ni Android Studio → `expo run:ios` & `expo run:android`. |
| **TypeScript 5.8** | Cero `any`, ciento por ciento autocompletado. |
| **NativeWind** | Tailwind CSS en componentes nativos: `&lt;View className="flex-row"&gt;`. |
| **Reanimated 3.17** | Gestos de 120 fps que se sienten como seda. |
| **EAS** | Builds en la nube con `eas build -p android --profile preview`. |
